# sms-bot

This sms-bot does an analysis of an inbound message to find keywords which trigger a response. The keywords in this version are 'pizza' and 'ice cream'.  

## Requirements

- Flask 
- Ngrok
- Python 3.6+
- Telnyx Python SDK
