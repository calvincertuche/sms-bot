# sms-bot

The sms-bot is an auto-responder that likes pizza and gelato. It receives a message, checks the message for keywords, and responds based on the keywords it did or did not receive. 

## Requirements

- Flask
- Ngrok
- Python 3.6+
- Telnyx Python SDK
